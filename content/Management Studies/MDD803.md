**Credits:** 3 (0-0-6)



#### Description 
The topic for the major project would be a generic one, pertaining to a management problem/issue related to any functional area viz. finance, operations, strategy, HR, marketing, IT, econometrics etc. Latest management & research tools and techniques would be applied in the analysis. The project would also be evaluated by an external expert. Each MBA project would be unique and the contents would be decided on the advice of the project supervisor. The project would need to focus on technology sector.
